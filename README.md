# InstantReply - Missed Call Textback SaaS

Never miss a lead again. Auto-SMS reply to missed calls for SMBs.

## Overview

InstantReply automatically sends customizable text messages when you miss a call, keeping potential customers engaged and converting missed calls into conversations.

## Features

- **Auto SMS Reply** - Instantly text back when a call is missed
- **Custom Messages** - Per-client customizable templates
- **Dashboard** - Track calls, messages, and response rates
- **Multi-Client** - Manage multiple business phone numbers
- **Analytics** - Conversion tracking and call insights
- **Billing** - Stripe-powered subscription management

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Database**: Supabase (PostgreSQL)
- **SMS/Voice**: Twilio
- **Payments**: Stripe
- **Styling**: Tailwind CSS
- **Deployment**: Vercel

## Quick Start

```bash
npm install
cp .env.example .env.local
# Fill in your env vars
npm run dev
```
